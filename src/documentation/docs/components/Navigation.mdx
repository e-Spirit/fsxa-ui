## Properties

<PropsTable>
  <PropsTableRow value="items" type="Array" required>
    Specify an array which should be rendered. An item has the following properties.
    <ul>
        <li>
            <span class="code">label</span> is the displayed name
        </li>
        <li>
            <span class="code">path</span> the path that a click on this item should lead to
        </li>
        <li>
            <span class="code">id</span> an id for the item
        </li>
        <li>
            <span class="code">children</span> an array of identical item elements (submenu items)
        </li>
    </ul>
  </PropsTableRow>
  <PropsTableRow value="handleNavClick" type="Function">
    Callback that will be triggered, when a link is clicked.
  </PropsTableRow>
  <PropsTableRow value="isActiveItem" type="Boolean" default="false">
    Call that determinates which path is activated.
  </PropsTableRow>
</PropsTable>
