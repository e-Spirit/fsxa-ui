## Properties

<PropsTable>
  <PropsTableRow value="items" type="Array" required>
    Specify an array which should be rendered. An item has the following properties.
    <ul>
        <li>
            <span class="code">label</span> is the displayed name
        </li>
        <li>
            <span class="code">path</span> the path that a click on this item should lead to
        </li>
        <li>
            <span class="code">id</span> an id for the item
        </li>
        <li>
            <span class="code">childPlacement</span> a string that can be either "left" or "right" or undefined. 
            Determines the direction the flyout menu for the item is rendered in.
        </li>
        <li>
            <span class="code">children</span> an array of item elements containing a label, path and an id.
        </li>
    </ul>
  </PropsTableRow>
  <PropsTableRow value="onItemClicked" type="Function">
    Callback that will be triggered, when a link is clicked. When this callback is called, the clicked item will be passed into it.
  </PropsTableRow>
  <PropsTableRow value="activeItemKeys" type="Array" default="false">
    An array of items that should be marked as active.
  </PropsTableRow>
</PropsTable>
