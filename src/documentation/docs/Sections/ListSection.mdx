## Properties

<PropsTable>
  <PropsTableRow value="items" required type="Array">
  Items that should be shown in the list (title: string, description: string, price: number, image: Image, url: string).
  </PropsTableRow>
  <PropsTableRow value="headline" type="String">
  Specify a headline for the item list. The separator will not be displayed, if no headline is specified.
  </PropsTableRow>
  <PropsTableRow value="filters" type="Array">
  A list of filters (key: string, value: string).
  </PropsTableRow>
  <PropsTableRow value="selectedFilters" type="Array">
  An array of keys of the selected filters. 
  </PropsTableRow>
  <PropsTableRow value="handleFilterChange" type="Function">
  A callback that will be invoked when the selected filters change. The new selected filters will be passed as an argument.
  </PropsTableRow>
  <PropsTableRow value="handleItemClick" type="Function">
  A callback that will be invoked when a list item is clicked. The clicked item will be passed as an argument.
  </PropsTableRow>
</PropsTable>
