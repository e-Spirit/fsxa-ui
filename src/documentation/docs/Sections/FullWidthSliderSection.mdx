## Slots

<SlotsTable>
  <SlotsTableRow value="title" prop="RenderedType">
  The title element to be displayed (and animated) on every slide.
  </SlotsTableRow>
  <SlotsTableRow value="teaser" prop="RenderedType">
    The teaser element to be displayed (and animated) on every slide.
  </SlotsTableRow>
  <SlotsTableRow value="media" prop="MediaType">
    The media element (the main content) for each slide. MediaType can e.g. be Image (with src, class, and opacity).
  </SlotsTableRow>
 <SlotsTableRow value="button" prop="Object">
    The button receives two attributes: <ul><li><span class="code">content: RenderedType;</span></li><li><span class="code">onClick: () => void;</span></li></ul>
  </SlotsTableRow>
  <SlotsTableRow value="arrowButton" prop="Object">
    Button elements to display for the larger resolutions. Attributes: <ul><li><span class="code">media</span> MediaType e.g. Image</li><li><span class="code">slideNumber</span> 0... n</li><li><span class="code">position</span> ('left' or 'right')</li></ul>
  </SlotsTableRow>
  <SlotsTableRow value="arrowButtonContent" prop="Object">
    This is an inner element to arrowButton, already in position. The object prop is the same.
  </SlotsTableRow>
  <SlotsTableRow value="controls" prop="RenderedType">
    The complete controls for the slider. The following props will be passed in:
    <ul>
      <li><span class="code">currentSlideIndex: number</span></li>
      <li><span class="code">nextSlideIndex: number</span></li>
      <li><span class="code">prevSlideIndex: number</span></li>
      <li><span class="code">showSlide: (index: number) => void</span></li>
    </ul>
  </SlotsTableRow>
  <SlotsTableRow value="stepper" prop="Object">
    The stepper is a section with the typical dots that becomes visible when the horizontal space for the slider is somewhat limited, and the arrows are no longer shown.
  </SlotsTableRow>
  <SlotsTableRow value="stepperStep" prop="Object">
    These are the stepper's individual dots. Obviously they're not rendered, when you replace the whole stepper (or even the complete controls).
  </SlotsTableRow>
</SlotsTable>
